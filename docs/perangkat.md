# Perangkat

Panduan ini menjelaskan langkah-langkah untuk menggunakan menu *Perangkat* di Aplikasi *Notifin*.

## Menambah Device

![perangkat](/img/perangkat/perangkat.png)

- Pada sidebar sebelah kiri, klik menu **Perangkat**.

![tambahdevice](/img/perangkat/tambahdevice.png)

- Dibagian atas tabel sebelah kanan klik tombol + Tambah Device.

![nmakantornmrwa](/img/perangkat/nmakantornmrwa.png)

- Setelah di klik akan muncul formulir tambah device baru.

- Isi formulir device baru, misalnya:

           **Nama Device (Wajib diisi)** ➜ ACCESSMEDIA

           **Nomor WhatsApp** ➜ 0812-3456-7890

![pilihdevice](/img/perangkat/pilihdevice.png)

- Klik kolom service, lalu pilih Test Service 1 (http://localhost:3000).

![simpan](/img/perangkat/simpan.png)

- Jika sudah diisi klik tombol **Simpan**.

![berhasilditambahkan](/img/perangkat/berhasilditambahkan.png)

- Setelah data disimpan, data device yang baru ditambahkan akan otomatis muncul dalam Daftar Perangkat.

----

## Fitur Connect

- Fitur Connect digunakan untuk menghubungkan perangkat baru (misalnya ponsel atau nomor WhatsApp) ke sistem Notifin agar bisa digunakan untuk mengirim atau menerima pesan otomatis.

![fiturconnect](/img/perangkat/fiturconnect.png)

- Klik tombol Connect.

![connect1](/img/perangkat/connect1.png)

- Setelah klik tombol Connect, akan muncul Scan QR Code, lalu hubungkan perangkat tertaut di WhatsApp dengan Scan QR Code tersebut. 

![connect2](/img/perangkat/connect2.png)

- Jika sudah berhasil connect, maka akan muncul keterangan berhasil terhubung.


## Fitur Recconnect

- Fitur Recconnect digunakan untuk menghubungkan kembali perangkat yang sudah terhubung ke sistem Notifin.

![reconnect](/img/perangkat/fiturreconnect.png)

- Klik tombol Recconnect.

![reconnectsukses](/img/perangkat/reconnectsukses.png)

- Jika sudah berhasil reconnect, maka akan muncul notifikasi reconnect suskes.

----

## Fitur Edit 

- Fitur Edit digunakan untuk mengubah nama device dan nomor WhatsApp.

![edit](/img/perangkat/edit.png)

- Klik tombol Edit.

![simpanperubahan](/img/perangkat/simpanperubahan.png)

- Jika sudah diubah klik tombol **Simpan Perubahan**.

![berhasiledit](/img/perangkat/berhasildiperbarui.png)

-  Jika sudah berhasil di edit, maka akan muncul notifikasi device berhasil diperbarui.

----

✅ Selesai! Pengaturan menu Perangkat berhasil dilakukan.

# Pengenalan

Notifin adalah sebuah aplikasi notifikasi yang memungkinkan pengguna untuk mengirim dan mengelola pesan secara mudah, cepat, dan efisien.
Dengan Notifin, kamu dapat mengatur perangkat pengirim pesan, mengelola kontak penerima, serta memantau status pengiriman pesan dengan tampilan yang sederhana dan intuitif.

![intro](/img/intro.png)

Dalam panduan ini, kita akan mempelajari cara menggunakan Notifin untuk membantu proses komunikasi dan pengiriman notifikasi di organisasi atau bisnis kamu.
Panduan ini juga akan menjelaskan fitur-fitur utama yang tersedia di Notifin dan bagaimana cara menggunakannya secara efektif untuk memaksimalkan fungsinya.
